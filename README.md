# React Counter App

A simple counter application built with React.

## Features

- Increase and decrease the number  
- Minimum value: 0  
- Maximum value: 20  
- Warning message when value is too high or too low  

## Technologies

- React  
- JavaScript  
- CSS  

## How to Run

1. Download or clone the repository  
2. Open the project folder  
3. Install dependencies:
   npm install  
4. Start the app:
   npm start  

## What I Practiced

- useState hook  
- Event handling  
- Conditional rendering  
- Basic logic and limits
